Discord Bot "Game News Bot"

(Game News Bot Wiki)

Developed a C# Discord bot using DSharpPlus that retrieves game news from the Steam Web API and posts updates to designated channels. Implemented asynchronous API requests with robust null‑checking and partial‑response handling to prevent failures when the API returns incomplete or empty data. Added structured command modules, including administrator commands (Ban, Kick, etc.), and built a flexible foundation for future automated sync/polling features. Designed the bot to support manual triggers for fetching the latest news while maintaining clean architecture, async/await patterns, and defensive programming for reliability.


Team ToolBox - Internal Automation Application

Developed an internal C#/.NET desktop application that automates SQL and PostgreSQL script generation from user‑entered parameters. Implemented MVVM architecture, structured input validation, and robust null‑checking to support partial or empty input, enabling users to generate base templates without full parameter sets. Built a modular script‑templating engine, automatic directory creation, safe file‑write operations, and one‑click access to the generated scripts folder. Designed a clean UI with themes prototyped using Claude AI. Followed SOLID principles, async patterns, and internal logging to ensure reliability and maintainability.
