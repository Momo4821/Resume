  • Discord Bot "Game News Bot"

    (Game News Bot Wiki)

    Developed a C# Discord bot using DSharpPlus that retrieves game news from the Steam Web API and posts updates to designated channels. Implemented asynchronous API requests with robust null‑checking and partial‑response handling to prevent failures when the API returns incomplete or empty data. Added structured command modules, including administrator commands (Ban, Kick, etc.), and built a flexible foundation for future automated sync/polling features. Designed the bot to support manual triggers for fetching the latest news while maintaining clean architecture, async/await patterns, and defensive programming for reliability.
